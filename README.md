# numeroPerfeitoRange
Shell script que mostra os números perfeitos encontrados num range de números informados na execução. <br>
Ao final mostra o total de números perfeitos encontrados.

# Otimizações de código

* Analisa até o divisor ser no máimo sua metade: <br>
  Ex: Número informado = 10 <br>
  Ele checará até o 5 no máximo.
